How It Works

Three simple steps to intelligent conversations

1

Purchase & Install

Purchase the plugin and install it on your WordPress site. After purchase, you'll receive the plugin files and license key.

2

Add Your OpenAI API Key

Important: You need an OpenAI API key to use this plugin. Get your key from platform.openai.com/api-keys. All API requests (embeddings and chat) will use your key and count towards your OpenAI usage/budget.

3

Process Content & Start Chatting

Process your website content to create embeddings. Your chatbot is ready! Visitors can ask questions and get intelligent answers based on your actual website content.
